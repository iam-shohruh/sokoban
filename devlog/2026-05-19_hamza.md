# Devlog

## Date
2026-05-19

## Features Added

### Interactive Solver Menu
Added a terminal menu system to the C version of the Sokoban solver.

Users can now:
- Choose between `Microban` and `XSokoban`
- Choose the solving algorithm:
  - BFS
  - A*
- Choose a specific level number

Input validation was added to prevent crashes from invalid inputs.

---

### Solver Animation
Implemented real-time visualization of the computer solving the puzzle.

The board now updates step-by-step while the solver runs, showing:
- Current board state
- Current move count
- Push direction

This replaced the old behavior where only the final result and timing were shown.

---

### Deadlock Detection Improvements
Updated the Python deadlock detection logic using:
- Corner deadlock checks
- 2x2 square deadlock checks
- Wall trap detection
- Goal-aware deadlock filtering

This reduced unnecessary searches and improved solving efficiency.

---

### Dataset Support
Fixed dataset loading and level counting.

Current counts:
- Microban: 155 levels
- XSokoban: 90 levels
